# meterm-backend

### Steps to run the app.

#### Prerequisites
- ```Node.js```
- ```Npm or Yarn```
- ```MongoDB```

<br>

Clone the repo
```bash
git clone https://github.com/divinirakiza/meterm-backend
```

Install Dependencies
```
npm i
```
or
```
yarn
```

Run the server
```
npm run dev
```
or
```
yarn dev
```

Run tests
```
npm run test
```
or
yarn test
```
